Some of the Courses covered in this academic year

Computer Science Principles and Programming

This course covers fundemental concepts in CS along with a very popular introductory programming language called Python.

Introduction to data science

This course introduces methods for five key facets of an investigation: data wrangling, cleaning, and sampling to get a suitable data set; data management to be able to access big data quickly and reliably; exploratory data analysis to generate hypotheses and intuition; prediction based on statistical methods such as regression and classification; and communication of results through visualization, stories, and interpretable summaries.

Object Oriented Programming with Java

This course covers the object oriented concepts and fundamentals of Java programming

Algorithms and Data Structures

This course covers the essential information that we need to know about algorithms and data structures

Databases

This course covers the fundaments of DBMS
